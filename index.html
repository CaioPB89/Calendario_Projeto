Como usar o calendário

Mostrar feriados -> Os dados referentes a feriados são carregados para a aplicação diretamente, mas você pode marcar ou retirar a marcação para eles serem exibidos utilizando os botões abaixo do calendário.

Modificar eventos -> Para adicionar um evento, clique no dia para adicionar uma marcação, então clique na marcação para modificar o título, cor ou o remover.

Todos os eventos são salvos no armazenamento local. Caso queira limpar todos os eventos, use o botão "Limpar todos os eventos customizados"
